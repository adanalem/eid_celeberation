# eid_celeberation
A lightweight, zero-dependency web experience featuring custom particle physics, CSS keyframe animations, and a responsive glassmorphism UI. Built specifically to showcase creative front-end engineering for Eid-ul-Fitr 2026.
A premium, interactive web experience built with Vanilla JavaScript to celebrate Eid-ul-Fitr. This project features a "Royal Emerald & Gold" aesthetic, custom particle physics, and soulful animations—all contained within a single, lightweight file.


🚀 Features


Custom Particle Engine: A built-in sparkle system using the Element.animate() API for 60FPS performance.

Glassmorphism UI: Modern frosted-glass input card with gold-leaf accents.

Responsive Design: Fully optimized for Mobile, Tablet, and Desktop views.

Soulful UX: Includes a "Royal Intro" animation and a heartfelt Dua (blessing) generator.

🛠️ Technical Deep Dive
Animations: Utilizes CSS @keyframes for the floating lanterns and stars to keep the Main Thread free.

Visuals: Uses radial-gradient and backdrop-filter for a high-end depth effect.

Performance: Particle objects are automatically garbage-collected from the DOM after the animation finishes to prevent memory leaks.
